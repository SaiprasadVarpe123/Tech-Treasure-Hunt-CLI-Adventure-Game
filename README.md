🎮 TECH TREASURE HUNT: Gameplay Overview

The Tech Treasure Hunt is a command-line based interactive adventure game designed for a live college tech fest event. Participants progress through multiple levels by solving riddles and unlocking clues that lead them closer to the final treasure.

🔄 How It Works

Participants run the Python program in the terminal.

A riddle is displayed on the screen.

Based on the clue, participants visit a physical location within the campus.

At the location, they find a hidden password.

The password is entered into the program to unlock the next challenge.

This process continues through multiple levels.

After completing all challenges, the final clue reveals the treasure location.

🧩 Game Structure

Multi-level sequential progression

Password validation system

Interactive terminal interface with colored output

Immediate feedback for correct/incorrect answers

Final victory screen upon completion
