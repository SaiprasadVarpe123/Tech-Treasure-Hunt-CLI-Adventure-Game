🎮 TECH TREASURE HUNT: Gameplay Overview

The Tech Treasure Hunt is a command-line based interactive adventure game designed for a live college tech fest event. Participants progress through multiple levels by solving riddles and unlocking clues that lead them closer to the final treasure.

🔄 How It Works

Participants run the Python program in the terminal.<br>
A riddle is displayed on the screen.<br>
Based on the clue, participants visit a physical location within the campus.<br>
At the location, they find a hidden password.<br>
The password is entered into the program to unlock the next challenge.<br>
This process continues through multiple levels.<br>
After completing all challenges, the final clue reveals the treasure location.


🧩 Game Structure

Multi-level sequential progression.<br>
Password validation system.<br>
Interactive terminal interface with colored output.<br>
Immediate feedback for correct/incorrect answers.<br>
Final victory screen upon completion
